# Rodel Reader - Feedback Repository

Rodel Reader is a modern Windows desktop reader based on WinUI 3.<br/>

## About This Repository

[简体中文](./README.md) | **English**

This is the public feedback repository for **Rodel Reader**, used to collect user feedback and issue reports.

The application itself is closed-source. This repository is only for:

- Submitting bug reports
- Submitting feature requests
- Discussing documentation issues

## Links

- **Documentation**: [https://reader.richasy.net](https://reader.richasy.net)
- **Microsoft Store**: [Download](https://apps.microsoft.com/detail/9pfzckrhw0bc)

## How to Submit Feedback

Please submit your feedback via [Issues](https://github.com/Richasy/Rodel.Reader.Public/issues/new/choose). We provide the following templates:

- **Bug Report** - Report issues encountered while using the app
- **Feature Request** - Suggest new features or improvements
- **Document Change** - Suggestions or issues about project documentation
