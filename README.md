## 关于此仓库

**简体中文** | [English](./README.en.md)

这是**小幻阅读**的公开反馈仓库，用于收集用户反馈和问题报告。

应用本体为闭源项目，此仓库仅用于：

- 提交 Bug 报告
- 提交功能建议
- 讨论文档相关问题
